
# Amazon Superstore Sales Dashboard

## Project Overview
This project is an interactive Sales Dashboard developed using Microsoft Power BI. It helps analyze business performance through KPIs and interactive visualizations.

## Features
- KPI Cards (Sales, Profit, Orders, Customers)
- Sales by Category
- Profit by Segment
- Country-wise Sales (Treemap)
- Top 10 Products by Sales
- Interactive Slicers (Date, Region, Category, Segment)

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

## Files Included
- Amazon Superstore Sales Report.pdf

- Dashboard Screenshot

- Better Sales Analysis
- Faster Decision Making
- Interactive Business Reporting
